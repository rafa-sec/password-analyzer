# Password Analyzer

## What it does

This tool analyzes the strength of a password and estimates how resistant it is against brute-force attacks.

## Why I built this

I have a deep passion for cybersecurity and wanted to understand how password strength translates into real-world attack scenarios, especially brute-force and dictionary attacks.

## Features

* Length analysis ✅(Done)
* Character diversity analysis (lowercase, uppercase, digits, symbols) ✅(Done)
* Entropy estimation ✅(Done + security rules V.3)
* Feedback system ✅(Done)
* Pattern detection
* Brute-force attack time simulation
